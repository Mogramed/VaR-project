# Figma Foundations

## Visual Thesis
- Mood: calm dark trading desk, dense but breathable
- Material: graphite, deep slate, copper accent, restrained glow
- Energy: premium operator workspace rather than card-heavy SaaS

## Type System
- UI hierarchy: `Manrope`
- Numeric and technical layer: `IBM Plex Mono`

## Color Tokens
- Background: `#08090b`
- Surface: `rgba(16, 20, 26, 0.78)`
- Surface strong: `rgba(21, 26, 34, 0.96)`
- Accent: `#d89b49`
- Success: `#5fd4a6`
- Warning: `#f2b45d`
- Danger: `#f27575`

## Layout Rules
- Landing hero is full-bleed and poster-like
- Desk uses three regions:
  - left navigation rail
  - central analysis workspace
  - right inspector and governance rail
- Avoid dashboard mosaics and stacked mini-cards

## Motion Rules
- Hero reveal on load
- Subtle section transitions
- No ornamental micro-motion
- Charts animate only enough to improve readability

## Page Inventory
- Landing EN/FR
- Desk overview
- Models
- Attribution
- Capital
- Decisions
- Simulation
- Reports
